# GitHub Pages setup

Aktualny uklad repozytorium zaklada publikacje z katalogu glownego repo, a nie z folderu `docs`.

## Ustawienia

1. Push this repository to GitHub.
2. Open `Settings` in the repository.
3. Go to `Pages`.
4. Set:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/(root)`
5. Save.
6. Wait for the first deployment URL.

## Before enabling Pages

Replace all placeholders:

- `[UZUPELNIJ: ...]`
- `[FILL IN: ...]`

Especially:

- privacy contact email,
- account deletion email or form URL,
- controller details,
- data retention periods,
- payment information after Premium launch.
