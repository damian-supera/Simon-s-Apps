Jak z tego korzystac

  • Dla kazdej aplikacji utworz osobny katalog, np. /dateup albo /app-2.
  • W sklepie podawaj URL do konkretnej aplikacji, a nie tylko do strony glownej.
  • Zostaw wspolne style i skrypt w katalogu glownym, zeby latwiej utrzymywac calosc.

How to use this structure

  • Create a separate folder for each app, such as /dateup or /app-2.
  • In the stores, use the URL for the specific app rather than only the home page.
  • Keep shared styles and the shared script in the root folder to make maintenance easier.

GitHub Pages setup: GitHub -> repo -> Settings -> Pages -> Source: Deploy from a branch -> Branch: main -> Folder: /(root).
